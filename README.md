# Snake Game
Классическая аркадная игра **"Змейка"** созданная на **Python** с библиотекой **Pygame**

## Описание
Игрок **управляет змейкой собирая еду** и **избегая столкновения с самим собой** и **с краяами окна**

## Демо
![photo1](snake_photo1.png)
![photo2](snake_photo2.png)

**Управление:**  
  - W: Вверх
  - S: Вниз
  - A: Влево
  - D: Вправо

## Технологии
- **Python 3.x**
- **Pygame**

## Установка
1. Установите библиотеку **Pygame** если она еще не установлена:
   ```bash
   pip install pygame
   ```
2. Склонируйте или скачайте этот репориторий:
   ```bash
   git clone https://github.com/AmirCode228/Snake-Game.git
   ```
3. Перейдите в папку проекта:
   ```bash
   cd Snake-Game
   ```
4. Запустите main.py:
   ```bash
   python main.py
   ```
